    前言

    还是一样，一开始先展示一下我的个人博客

    前言：本教程只适用于Windows系统搭建，Linux或Macos后续会更新。

    搭建准备工作

    因为hexo是一款基于Node.js的静态博客框架，依赖少而且易于使用和操作，可以很简易的生成静态网页托管到github或者码云上。具体的详情大家可以进入hexo官网进行查看

    安装Git

    如果没有github的账号就先去创建，在创建同时为了节约时间，可以先去Git官网将Git下载安装。

    安装完成后，可以用Git安装路径文件夹下的gitbash来进行命令行的创造，不需要用自带的cmd。

    然后再安装Node.js

    进入 Node.js官网 ，选择LTS版本即可。

    安装完成后，Win+R 打开命令行，输入

    Code

    检查一下有没有安装成功，若成功，即有相应的版本号生成

    核心工作

    安装hexo

    准备工作完成后，在自己空余多的硬盘里创建一个文件夹，cd到这个文件夹，或者在文件夹中右键点击gitbash命令打开。
    输入命令

    npm install -g hexo cli

    安装完成后依旧用`hexo -v·检查一下版本。

    至此安装工作全部完成。

    初始化hexo

    hexo init myblog

    这里的myblog是你文件夹的名字
    ，接下来

    cd myblog

    npm install

    安装完成后，在你的博客目录下会产生以下文件：

    • node_modules: 依赖包
    • public：存放生成的页面
    • scaffolds：生成文章的一些模板
    • source：用来存放你的文章
    • themes：主题
    • _config.yml: 博客的配置文件

    hexo g //g是缩写，generate，生成静态界面

    hexo server //启动hexo服务

    在浏览器输入localhost:4000，即可看到博客，如果懒得输入，点这里。

    使用ctrl + c可以把服务关闭。
